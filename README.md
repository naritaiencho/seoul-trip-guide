# SEOUL for Two — 2泊3日ふたりの韓国ガイド

2026.7.4(土)昼 〜 7.6(月)昼過ぎ、30代の男女の「初めての韓国」専用ガイドサイト。弘大(ホンデ)を拠点に、渡韓40回超の妹のお墨付き情報を軸に構成。

## 起動

```bash
npm install
npm run dev    # http://localhost:5173
npm run build  # dist/ に静的ビルド
```

## 構成

- `src/data/` — リサーチ済みコンテンツ（エリア8・グルメ34軒・専門店通り6・観光13スポット・ホテル8軒・モデルプラン2種・Tips）
- `src/components/` — セクションコンポーネント（妹のお墨付き→グルメ図鑑→専門店ストリート→観光→…）
- `src/store/TripStore.tsx` — お気に入り・制覇・ホテル決定・持ち物の状態管理（localStorage永続化）
- `public/img/` — GPT Image 2（codex exec）で生成した統一スタイルのイラスト（WebP・25枚）
- `research/` — Webリサーチの一次情報メモ（`imouto.md`＝妹おすすめ第1弾、`imouto-batch2.md`＝第2弾の根拠）

## インタラクティブ機能

- ♥ **行きたい！** / ✅ **行った！** — グルメ・観光カードにマーク（端末のlocalStorageに保存）
- 🏨 **ここに泊まる** — ホテル決定でカード強調＋ヒーローの搭乗券に宿名が反映
- 📖 **ふたりのしおり**（#shiori） — お気に入り自動集約・制覇率ゲージ・テキストコピー共有
- 🎰 **今夜どこ行く？ルーレット** — お気に入り2軒以上でお気に入り限定抽選に切替
- ☀️ **ソウル天気** — Open-Meteo API。旅行日が予報範囲（約2週間前〜）に入ると自動で7/4〜6を表示
- 💱 **ウォン⇄円電卓**・🧳 **持ち物チェックリスト**（プリセット13項目＋自由追加）
- 旅行当日はモデルプランに **TODAY** バッジが自動表示

※ マークは端末ごとの保存（共有DBなし）。2人で同期したい場合は「しおりをコピー」でLINE共有する運用。

## コンテンツの注意

料金・営業時間は2026年6月時点のWebリサーチ値。訪問直前にNAVER Map／CATCHTABLE等で最新情報の確認を推奨。日程に直結する定休に注意：

- **クンソンチッ**（エゴマユッケジャン）: 日曜定休・土曜15時まで → 月曜朝が本命。
- **アイラブ新堂洞**（トッポギ鍋）: 第1・第3月曜定休 → 最終日 **7/6(月)＝第1月曜は休みの可能性大**。トッポギは7/4・7/5に。
- **꿉당/ソムンナンカムジャタン等の人気店**: 予約アプリ **CATCHTABLE（グローバル版・日本語対応）** で順番待ち推奨。予約金が発生する店はキャンセル規定に注意。
- **タクシー**: 黒い大型バン「コールバン」や日本語客引きの白タクは避け、Kakao T／Uberで配車するのが安全。
