# SEOUL for Two — 2泊3日ふたりの韓国ガイド

2026.7.4(土)昼 〜 7.6(月)昼過ぎ、35歳男性＋同世代女性の「初めての韓国」専用ガイドサイト。

## 起動

```bash
npm install
npm run dev    # http://localhost:5173
npm run build  # dist/ に静的ビルド
```

## 構成

- `src/data/` — リサーチ済みコンテンツ（エリア・グルメ30軒・観光12スポット・ホテル8軒・モデルプラン2種・Tips）
- `src/components/` — セクションコンポーネント
- `src/store/TripStore.tsx` — お気に入り・制覇・ホテル決定・持ち物の状態管理（localStorage永続化）
- `public/img/` — GPT Image 2（codex exec）で生成した統一スタイルのイラスト20枚（WebP）
- `research/` — Webリサーチの一次情報メモ（営業時間・価格の根拠）

## インタラクティブ機能

- ♥ **行きたい！** / ✅ **行った！** — グルメ・観光カードにマーク（端末のlocalStorageに保存）
- 🏨 **ここに泊まる** — ホテル決定でカード強調＋ヒーローの搭乗券に宿名が反映
- 📖 **ふたりのしおり**（#shiori） — お気に入り自動集約・制覇率ゲージ・テキストコピー共有
- 🎰 **今夜どこ行く？ルーレット** — お気に入り2軒以上でお気に入り限定抽選に切替
- ☀️ **ソウル天気** — Open-Meteo API。旅行日が予報範囲（約2週間前〜）に入ると自動で7/4〜6を表示
- 💱 **ウォン⇄円電卓**・🧳 **持ち物チェックリスト**（プリセット13項目＋自由追加）
- 旅行当日はモデルプランに **TODAY** バッジが自動表示

※ マークは端末ごとの保存（共有DBなし）。2人で同期したい場合は「しおりをコピー」でLINE共有する運用。

## コンテンツの注意

料金・営業時間は2026年6月時点のWebリサーチ値。訪問直前にNAVER Map等で最新情報の確認を推奨。
特にクンソンチッ（エゴマユッケジャン）は **日曜定休・土曜15時まで** なので月曜朝が本命。
